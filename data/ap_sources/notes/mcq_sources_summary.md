## 2. Course at a Glance 要点

### 考试相关单元与权重（计入 AP 考试）

| 单元 | 名称 | 建议周数 | 考试权重 |
|------|------|----------|----------|
| Unit 1 | Polynomial and Rational Functions | 6–8 | **30–40%** |
| Unit 2 | Exponential and Logarithmic Functions | 6–9 | **27–40%** |
| Unit 3 | Trigonometric and Polar Functions | 7–10 | **30–35%** |
| Unit 4 | Parameters, Vectors, Matrices | 7 | **0%**（不考） |

### 数学实践（贯穿出题）

1. Procedural and Symbolic Fluency  
2. Multiple Representations（图、表、式、情境）  
3. Communication and Reasoning  

### Unit 1 主要主题（节选）

- 1.1–1.3：Change in Tandem、Rates of Change、线性与二次变化率  
- 1.4–1.6：多项式变化率、复零点、**End Behavior**  
- 1.7–1.10：有理函数端点行为、零点、**竖直渐近线、holes**  
- 1.11–1.14：等价表示、变换、**模型选择与构造**  

### Unit 2 主要主题（节选）

- 2.1–2.2：数列、线性与指数变化  
- 2.3–2.6：指数函数、运算、**情境建模**、模型比较  
- 2.7–2.15：复合、反函数、对数、方程、**semi-log**  

### Unit 3 主要主题（节选）

- 3.1–3.7：**周期现象**、正弦余弦、图像、**正弦建模与变换**  
- 3.8–3.12：正切、反三角、恒等式等价  
- 3.13–3.15：**极坐标**、极函数图像、极函数变化率  

### Progress Check MCQ 规模（每段约 18–24 道 MCQ + 2 FRQ）

便于理解官方题库密度，非本项目自动生成目标。

## 4. 对 style pattern 的启示（摘要）

1. **题干句式**：高频 “Which of the following …” / “Which of the following statements is true …” / “Based on this information …” / “could define / could be true”。  
2. **表征多样**：单题常混合图、表、情境、符号；Part B 偏重表与回归、图上的变化率与累积。  
3. **干扰项逻辑**：模型类型混淆（线性/指数/二次）、参数与变换混淆、hole vs 渐近线 vs 截距、残差高估/低估、弧度/象限/共终边角。  
4. **技能命名**：生成 API 的 `skill` 建议使用 Course at a Glance 主题名，或通过 `pattern_loader` 别名映射 slug（见 `ap_mcq_style_observations.md`）。
